# Como o app funciona?

1. Baixe um áudio(.mp3)
2. Abra o aplicativo e selecione a opção desejada (Ativar o áudio ao conectar o carredor e/ou atingir uma porcentagem selecionada)
3. Navegue até onde você baixou o áudio e selecione-o

## Pronto o app já vai estar funcionando
